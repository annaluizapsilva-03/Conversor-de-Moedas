# Conversor-de-Moedas

Uma aplicação web desenvolvida com HTML, CSS e JavaScript que permite converter valores entre Real (BRL), Dólar (USD) e Euro (EUR) por meio de uma interface moderna e intuitiva.

Este projeto foi desenvolvido com o objetivo de praticar conceitos de lógica de programação, manipulação do DOM e criação de interfaces responsivas utilizando apenas tecnologias front-end.

✨ Funcionalidades
💰 Conversão entre Real, Dólar e Euro
🔄 Seleção da moeda de origem e destino
⚡ Cálculo instantâneo
📱 Interface moderna e responsiva
🎨 Design com efeitos visuais e gradientes
🛠️ Tecnologias Utilizadas
HTML5
CSS3
JavaScript
